# Game Outline

## Basic overview

The game is a 3d planet customiser right now
the player can move around using wasd and q e
the player can open the chat with 'Enter' and type things
Commands:
'/CHANGENAME <new player name>'
'/DEBUG' activate debug on server
'/RECONNECT' reconnect to server
'/GUI' open planet customisation menu

the player can fly around, type in chat, or customise the planet

## Libraries used:

1. Open GL (rendering framework)
2. GLFW (wrapper for Open GL to improve OPEN GL calls)
3. GLM (math library for open GL)
4. IMGUI (deccoupled GUI realtime)
5. Freetype (character rasterization for rendering of fonts)
